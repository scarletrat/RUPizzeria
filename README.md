## Overview

A JavaFX-based pizza ordering system for RU Pizzeria. The system allows store staff to take, manage, and export customer orders for Chicago-style and New York-style pizzas. This project implements a multi-view GUI application using JavaFX and the **Abstract Factory** design pattern to manage pizza orders. Staff can build custom orders, review them, place them, and export all orders to a text file.

## Features

- Order Chicago-style or New York-style pizzas in small, medium, or large
- Three specialty pizzas (Deluxe, BBQ Chicken, Meatzza) with fixed toppings
- Build Your Own pizza with up to 7 customizable toppings
- Dynamic running subtotal as toppings are added/removed
- Multi-pizza orders — add or remove pizzas before placing
- Order review with itemized list, subtotal, 6.625% NJ sales tax, and order total
- Browse all placed orders and cancel any order
- Export all orders to a text file
- Unique order numbers auto-generated by the system


## Design Patterns

- **Abstract Factory** — `PizzaFactory` interface implemented by `ChicagoPizza` and `NYPizza` to create pizza objects. Only factory instances may instantiate pizza subclasses.
- **MVC** — Each `.fxml` view has a dedicated `*Controller.java`; model classes are kept separate from UI logic.

## How to Run

1. Ensure **JDK 17+** and **JavaFX SDK** are installed.
2. Clone the repository:
   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   ```
3. Open in **IntelliJ IDEA** and configure JavaFX VM options:
   ```
   --module-path /path/to/javafx-sdk/lib --add-modules javafx.controls,javafx.fxml
   ```
4. Run the main application class.
